# Workmate SDK - Android

**Field-Force Management SDK for Android Applications**

Powered by India's most comprehensive mapping and location intelligence platform — Mappls.

Workmate SDK is built for organisations that operate a distributed field workforce — sales teams, service technicians, delivery agents, healthcare field workers, or any role where people work outside a fixed office. From the moment an agent checks in for the day, through every client visit and task completed, to the end of their shift — Workmate tracks, coordinates, and logs it all.
