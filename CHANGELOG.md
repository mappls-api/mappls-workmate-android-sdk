## 1.3.0 — 27 Feb, 2026

### Added
- **Task Check-In / Check-Out** — Previously the system supported only Client Check-In / Check-Out. Task-level check-in and check-out are now fully supported.
- **Separate Check-In and Check-Out functions** — A single combined function previously handled both operations. Dedicated `checkIn` and `checkOut` functions have been introduced for both tasks and clients.
- **New task-related functions:**
  - Get Task List
  - Get Assigned Tasks
- **New client-related function:**
  - Get Client List
- **Improved error handling** — All errors are now returned as structured error objects containing:
  - Error Code
  - Error Message
- **Automatic route optimization** — User-assigned plans linked to locations are now automatically arranged using an optimized route to improve field efficiency.
